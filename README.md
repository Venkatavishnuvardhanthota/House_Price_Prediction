 # House Price Prediction

**Internship Project — Week 1 | XYlofy AI**

## Problem Statement
Build a regression model to predict house prices based on property features (size, rooms, location, age) and identify which features most influence price.

[Housing Prices Dataset — Kaggle](https://www.kaggle.com/datasets/yasserh/housing-prices-dataset)
